다담 SEO 체크 v2.0 - 무료 진단 + 유료 상세 보고서
=====================================================

[현재 구현된 흐름]
1. 사용자가 홈페이지 주소와 목표 키워드를 입력합니다.
2. 서버가 홈페이지 첫 화면의 실제 HTML을 분석합니다.
3. 종합 점수, 요약, 핵심 문제 3개는 무료로 보여줍니다.
4. 나머지 상세 결과는 서버의 storage/reports 폴더에만 보관합니다.
5. 사용자가 결제하면 서버에서 결제 금액과 주문번호를 확인합니다.
6. 토스페이먼츠 승인 API 결과가 DONE일 때만 전체 보고서를 엽니다.
7. 상세 보고서는 인쇄 또는 브라우저 PDF 저장이 가능합니다.

[서버 조건]
- PHP 8.1 이상
- PHP cURL
- PHP DOM/XML
- PHP mbstring
- HTTPS (실제 토스 결제 시 필수)
- storage 폴더 쓰기 권한

[가장 먼저 테스트하는 방법]
1. 압축을 풀고 파일 전체를 PHP 호스팅 웹 루트에 올립니다.
2. https://내도메인/health.php 에 접속합니다.
3. 모든 항목이 정상인지 확인합니다.
4. config.php는 기본값이 payment_mode => demo 입니다.
5. 홈페이지 주소를 분석한 뒤 상세 보고서 결제 버튼을 누릅니다.
6. 데모 결제 완료 버튼을 누르면 실제 결제 없이 상세 결과가 열립니다.

[토스페이먼츠 테스트 결제로 바꾸기]
config.php에서 아래 항목을 수정합니다.

'payment_mode' => 'toss',
'toss' => [
    'client_key' => 'test_ck_로 시작하는 API 개별 연동 클라이언트 키',
    'secret_key' => 'test_sk_로 시작하는 짝이 맞는 시크릿 키',
],

- 클라이언트 키와 시크릿 키는 반드시 같은 세트여야 합니다.
- 시크릿 키는 절대 자바스크립트, HTML, 공개 GitHub에 넣지 마세요.
- 라이브 전환 시 test 키를 live 키로 함께 교체합니다.
- 실제 결제는 토스페이먼츠 계약 및 카드사 심사 완료 후 가능합니다.

[가격 변경]
config.php의 아래 값을 수정합니다.
'report_price' => 4900,

[무료 공개 문제 개수 변경]
'free_check_count' => 3,

[사업자 정보]
config.php의 business 항목을 실제 사업자 정보로 반드시 변경하세요.
이용약관, 개인정보처리방침, 환불정책은 기본 초안이므로 실제 사업 방식과 법률 검토에 맞게 수정해야 합니다.

[보안 구조]
- 상세 결과는 결제 전 HTML에 포함하지 않습니다.
- 보고서 번호는 64자리 무작위 토큰입니다.
- 결제 금액은 successUrl 값만 믿지 않고 서버 저장 금액과 비교합니다.
- 승인 API는 서버에서 시크릿 키로 호출합니다.
- 승인 상태 DONE, 주문번호, 승인 금액을 모두 확인한 뒤 보고서를 엽니다.
- storage 폴더는 .htaccess로 외부 접근을 차단합니다.
- 내부망 및 사설 IP 분석을 차단합니다.
- IP 기준 시간·일일 분석 횟수 제한이 있습니다.

[중요한 한계]
- 이 버전은 홈페이지 첫 화면 1개를 분석합니다. 사이트 전체 페이지 크롤링은 아닙니다.
- 구글 실제 검색 순위, Search Console, PageSpeed Insights, Core Web Vitals는 포함되지 않았습니다.
- Cloudflare, 로그인, 봇 차단, 자바스크립트 렌더링 사이트는 일부 분석이 실패할 수 있습니다.
- 파일 저장 방식 MVP입니다. 사용자가 많아지면 MySQL, 관리자 주문 관리, 자동 삭제 작업, 결제 취소 기능, 로그 모니터링을 추가해야 합니다.

[폴더 권한]
storage 폴더가 쓰기 불가로 나오면 호스팅 관리 화면 또는 FTP에서 쓰기 권한을 허용하세요.
Apache가 아닌 서버에서는 storage 폴더를 웹 루트 밖으로 옮기고 config.php의 storage_path를 수정하는 것이 안전합니다.

[테스트 완료 후]
- health.php 삭제 또는 접근 제한
- config.php 실제 정보 입력
- 이용약관/개인정보/환불정책 검토
- HTTPS 적용
- 토스 테스트 결제 확인
- 토스 계약/카드사 심사 후 라이브 키 교체
